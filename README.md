# Space Invaders AI (Reinforcement Learning)

## Overview
This project implements a Space Invaders game environment with an AI agent trained using **Reinforcement Learning (Q-Learning)**. The AI learns to play the game and improve over episodes.

## Features
- Custom Space Invaders environment in Python (Pygame)
- AI agent using Q-Learning
- Visualization of reward vs episodes
- Fully configurable hyperparameters
